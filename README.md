# QR Code App
App that allow you working with QR Code using Kotlin MultiPlatform(KMP) and Compose Multiplatform for Android and Ios.

## Features

   ```text
   [✓] QR Code Generator
   [✓] Share QR Code
   [ ] Store Locally QR Code Data
   [ ] QR Code Scanner
   ```
