# QR Code App
A Simple App to generate, store and scan QR Code using Compose Multiplatform for Android and Ios.

## Features

   ```text
   [✓] QR Code Generator
   [✓] Share QR Code
   [ ] Save as Image in Gallery
   [ ] Store Locally QR Code Data
   [ ] QR Code Scanner
   ```
